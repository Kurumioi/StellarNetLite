# 03. StellarNet Lite 30分钟业务开发上手实践

> 本文不依赖任何 Demo。
> 
> 目标只有一个：让你从 0 开始，按框架基础能力，手动做出一套可运行的：
> 
> - 启动器
> - 全局模块
> - 房间业务组件
> - 客户端发送与监听
> - 服务端监听与回包

本文所有示例都只依赖框架本体：

- `Runtime`
- `Extensions`
- `StellarNetLiteGenerated`

不会依赖：

- `Samples/SocialDemo`
- Demo 的 `GameLauncher`
- Demo 的 `Panel`
- Demo 的 `Router`
- Demo 的任何 UI 逻辑

---

## 1. 做完本文后你应该能得到什么

做完本文后，你应该能独立完成下面这条完整链路：

1. 自己写一个启动器，启动 `Client` / `Server` / `Host`
2. 自己定义一组 `Global` 协议
3. 自己实现一组服务端全局模块和客户端全局模块
4. 自己定义一组 `Room` 协议
5. 自己实现一组服务端房间业务组件和客户端房间业务组件
6. 在客户端直接发送协议
7. 在客户端直接监听协议对象本身
8. 通过建房时的 `RoomDTO.ComponentIds` 把房间业务组件挂进房间
9. 用框架现成的默认房间流，跑通“登录 -> 建房 -> 进房 -> 房间业务消息”

本文故意不用复杂业务，只做两个最小案例：

- 全局模块案例：`Announcement`
- 房间组件案例：`RoomNotice`

它们足够小，但已经覆盖了最重要的开发主线：

`Launcher -> Shared 协议 -> Server 权威处理 -> Client 接收 -> 直接广播协议对象 -> 外部业务监听`

## 2. 先记住框架里最重要的 4 个角色

### 2.1 `StellarNetAppManager`

这是统一运行时入口。

它负责：

- 挂载并驱动传输层
- 读取并应用 `NetConfig`
- 创建 `ServerApp`
- 创建 `ClientApp`
- 装配自动注册出来的模块和房间组件

你自己写的启动器，本质上就是围绕它工作。

### 2.2 `ServerApp`

这是服务端主状态机。

它负责：

- 全局协议分发
- 会话管理
- 房间创建/销毁
- 给会话发全局消息

全局模块拿的是 `ServerApp`。

### 2.3 `Room`

这是服务端房间实例。

它负责：

- 房间成员管理
- 房间域协议分发
- 房间组件驱动
- 房间内广播

房间业务组件拿的是 `Room`。

### 2.4 `ClientApp` / `ClientRoom`

客户端主状态机是 `ClientApp`，当前房间实例是 `ClientRoom`。

客户端代码里要区分两类监听：

- 全局协议：`GlobalTypeNetEvent`
- 房间协议：`NetClient.CurrentRoom.NetEventSystem`

推荐做法很简单：

- 全局模块收到 `S2C_*` 后直接 `GlobalTypeNetEvent.Broadcast(msg)`
- 房间组件收到 `S2C_*` 后直接 `Room.NetEventSystem.Broadcast(msg)`

也就是说，外部业务层直接监听协议对象本身，不再额外包一层业务本地事件。

### 2.5 当前自动注册器对构造函数的要求

这一点非常重要，而且是“照着本文抄代码能不能直接跑起来”的关键。

当前自动生成的绑定器在创建模块和房间组件实例时，会按下面的签名注入上下文：

- 服务端全局模块：`public Xxx(ServerApp app)`
- 客户端全局模块：`public Xxx(ClientApp app)`
- 服务端房间组件：`public Xxx(ServerApp app)`
- 客户端房间组件：`public Xxx(ClientApp app)`

即使你当前示例代码里暂时还没用到 `_app`，也先不要删掉这几个字段和构造函数。

否则最常见的结果就是：

- 自动绑定代码编译不过
- 或者运行时无法按预期创建模块 / 组件实例

你可以把它理解成：当前框架不是“无参 new”，而是“由生成器统一注入 App 上下文”。

## 4. 第一步：从 0 写一个启动器

这个启动器只做框架基础工作，不做任何 Demo UI 初始化。

文件：

- `Assets/Scripts/MyGame/Runtime/BasicNetLauncher.cs`

代码：

```csharp
using System;
using StellarNet.Lite.Runtime;
using StellarNet.Lite.Shared.Infrastructure;
using UnityEngine;

namespace MyGame.Net
{
    public enum BasicNetMode
    {
        None,
        Client,
        Server,
        Host
    }

    public sealed class BasicNetLauncher : MonoBehaviour
    {
        [SerializeField] private StellarNetAppManager appManager;
        [SerializeField] private BasicNetMode netMode = BasicNetMode.Client;

        private void OnEnable()
        {
            StellarNetAppManager.OnClientStartedEvent += OnClientStarted;
            StellarNetAppManager.OnClientConnectedEvent += OnClientConnected;
            StellarNetAppManager.OnClientDisconnectedEvent += OnClientDisconnected;

            StellarNetAppManager.OnServerStartedEvent += OnServerStarted;
            StellarNetAppManager.OnServerStoppedEvent += OnServerStopped;
        }

        private void OnDisable()
        {
            StellarNetAppManager.OnClientStartedEvent -= OnClientStarted;
            StellarNetAppManager.OnClientConnectedEvent -= OnClientConnected;
            StellarNetAppManager.OnClientDisconnectedEvent -= OnClientDisconnected;

            StellarNetAppManager.OnServerStartedEvent -= OnServerStarted;
            StellarNetAppManager.OnServerStoppedEvent -= OnServerStopped;
        }

        private async void Start()
        {
            if (appManager == null)
            {
                Debug.LogError("BasicNetLauncher: appManager is null.");
                return;
            }

            NetConfig config = await NetConfigLoader.LoadRuntimeConfigAsync();
            appManager.ApplyConfig(config);

            switch (netMode)
            {
                case BasicNetMode.Client:
                    appManager.StartClient();
                    break;
                case BasicNetMode.Server:
                    appManager.StartServer();
                    break;
                case BasicNetMode.Host:
                    appManager.StartHost();
                    break;
            }
        }

        private void OnClientStarted()
        {
            Debug.Log("Client logic started.");
        }

        private void OnClientConnected()
        {
            Debug.Log("Client connected.");
        }

        private void OnClientDisconnected()
        {
            Debug.Log("Client disconnected.");
        }

        private void OnServerStarted()
        {
            Debug.Log("Server started.");
        }

        private void OnServerStopped()
        {
            Debug.Log("Server stopped.");
        }
    }
}
```

### 4.1 这个启动器依赖什么

你场景里至少要有一个 GameObject：

1. 挂 `StellarNetAppManager`
2. 再挂一个实现了 `INetworkTransport` 的传输层组件

例如：

- `KcpTransportProvider`
- `TcpTransportProvider`
- `UdpTransportProvider`

也就是说，启动器本身不直接处理网络包，它只负责：

- 加载配置
- 启动模式
- 订阅基础生命周期事件

## 5. 第二步：先定义一组全局协议

我们先做一个最小全局功能：公告请求。

文件：

- `Assets/Scripts/MyGame/Shared/Protocol/AnnouncementProtocols.cs`

代码：

```csharp
using StellarNet.Lite.Shared.Core;

namespace MyGame.Net
{
    [NetMsg(8201, NetScope.Global, NetDir.C2S)]
    public sealed class C2S_RequestAnnouncement
    {
        public string Content;
    }

    [NetMsg(8202, NetScope.Global, NetDir.S2C)]
    public sealed class S2C_AnnouncementSync
    {
        public string SenderName;
        public string Content;
    }
}
```

### 5.1 这一步的规则

一定要记住：

1. 协议定义放 `Shared`
2. 全局功能用 `NetScope.Global`
3. 客户端发服务端用 `NetDir.C2S`
4. 服务端回客户端用 `NetDir.S2C`

不要把协议类写进 `Client` 或 `Server`。

### 5.2 为什么这里没有手写 `Serialize / Deserialize`

本文这个案例故意只做低频业务消息，所以这里不实现 `ILiteNetSerializable`。

当前默认序列化器的规则是：

- 实现了 `ILiteNetSerializable` 的协议：走手写二进制
- 没实现的普通协议类：走 JSON

所以像本文里的：

- `Announcement`
- `RoomNotice`

这类低频教学消息，直接写普通字段就够了，先把开发主线跑通最重要。

如果后面你要做：

- 高频位置同步
- 动画同步
- 回放大块数据

再切到 `ILiteNetSerializable` 会更合适。

## 6. 第三步：写服务端全局模块

文件：

- `Assets/Scripts/MyGame/Server/Modules/ServerAnnouncementModule.cs`

代码：

```csharp
using StellarNet.Lite.Server.Core;
using StellarNet.Lite.Shared.Core;

namespace MyGame.Net
{
    [ServerModule("ServerAnnouncementModule", "公告模块")]
    public sealed class ServerAnnouncementModule
    {
        private readonly ServerApp _app;

        public ServerAnnouncementModule(ServerApp app)
        {
            _app = app;
        }

        [NetHandler]
        public void OnC2S_RequestAnnouncement(Session session, C2S_RequestAnnouncement msg)
        {
            if (session == null || msg == null)
            {
                return;
            }

            if (string.IsNullOrWhiteSpace(msg.Content))
            {
                return;
            }

            string senderName = string.IsNullOrWhiteSpace(session.AccountId)
                ? session.SessionId
                : session.AccountId;

            var syncMsg = new S2C_AnnouncementSync
            {
                SenderName = senderName,
                Content = msg.Content.Trim()
            };

            foreach (var kv in _app.Sessions)
            {
                Session target = kv.Value;
                if (target != null && target.IsOnline)
                {
                    _app.SendMessageToSession(target, syncMsg);
                }
            }
        }
    }
}
```

### 6.1 你需要理解的重点

全局模块的上下文是 `ServerApp`，不是 `Room`。

这意味着它适合做：

- 登录后大厅业务
- 公告
- 全局聊天
- 邮件入口
- 在线玩家列表

如果你的功能必须依附具体房间，就不要写在全局模块里。

## 7. 第四步：写客户端全局模块

文件：

- `Assets/Scripts/MyGame/Client/Modules/ClientAnnouncementModule.cs`

代码：

```csharp
using StellarNet.Lite.Client.Core;
using StellarNet.Lite.Client.Core.Events;
using StellarNet.Lite.Shared.Core;

namespace MyGame.Net
{
    [ClientModule("ClientAnnouncementModule", "公告模块")]
    public sealed class ClientAnnouncementModule
    {
        private readonly ClientApp _app;

        public ClientAnnouncementModule(ClientApp app)
        {
            _app = app;
        }

        [NetHandler]
        public void OnS2C_AnnouncementSync(S2C_AnnouncementSync msg)
        {
            if (msg == null || _app == null)
            {
                return;
            }

            GlobalTypeNetEvent.Broadcast(msg);
        }
    }
}
```

### 7.1 为什么这里直接广播协议对象

推荐做法是：

- 不额外定义 `Local_AnnouncementReceived`
- 直接广播 `S2C_AnnouncementSync`

这样外部业务层只需要记住一套名字：

- 协议名

而不是两套：

- 协议名
- 本地转发事件名

## 9. 第六步：写服务端房间业务组件

文件：

- `Assets/Scripts/MyGame/Server/Components/ServerRoomNoticeComponent.cs`

代码：

```csharp
using StellarNet.Lite.Server.Core;
using StellarNet.Lite.Shared.Core;

namespace MyGame.Net
{
    [RoomComponent(930, "RoomNotice", "房间通知组件")]
    public sealed class ServerRoomNoticeComponent : ServerRoomComponent
    {
        private readonly ServerApp _app;

        public ServerRoomNoticeComponent(ServerApp app)
        {
            _app = app;
        }

        [NetHandler]
        public void OnC2S_SendRoomNotice(Session session, C2S_SendRoomNotice msg)
        {
            if (Room == null || session == null || msg == null)
            {
                return;
            }

            if (string.IsNullOrWhiteSpace(msg.Content))
            {
                return;
            }

            Room.BroadcastMessage(new S2C_RoomNoticeSync
            {
                SenderSessionId = session.SessionId,
                Content = msg.Content.Trim()
            });
        }
    }
}
```

### 9.1 为什么这里拿的是 `Room`

因为房间业务组件本来就属于某个房间实例。

它适合处理：

- 房间聊天
- 房间通知
- 房间准备
- 房间战斗
- 房间交互

它天然有：

- `Room`
- 当前房间成员
- 当前房间状态

## 11. 第八步：写一个客户端调试面板脚本

                RoomConfig = new RoomDTO
                {
                    RoomName = "MyFirstRoom",
                    MaxMembers = 8,
                    EnableReplayRecording = false,
                    Password = string.Empty,
                    CustomProperties = new Dictionary<string, string>(),
                    ComponentIds = new[]
                    {
                        ComponentIdConst.RoomSettings,
                        ComponentIdConst.RoomNotice
                    }
                }
            });
        }

        public void JoinRoomById()
        {
            if (string.IsNullOrWhiteSpace(joinRoomId))
            {
                AppendLog("加入房间跳过: 房间 Id 为空。");
                return;
            }

            NetClient.Send(new C2S_JoinRoom
            {
                RoomId = joinRoomId.Trim(),
                Password = string.Empty
            });
        }

        public void ToggleReady()
        {
            ClientRoomSettingsComponent settings = GetRoomSettings();
            if (settings == null || NetClient.Session == null)
            {
                AppendLog("切换准备状态跳过: RoomSettings 尚未就绪。");
                return;
            }

            bool currentReady = false;
            if (!string.IsNullOrEmpty(NetClient.Session.SessionId) &&
                settings.Members.TryGetValue(NetClient.Session.SessionId, out MemberInfo self) &&
                self != null)
            {
                currentReady = self.IsReady;
            }

            NetClient.Send(new C2S_SetReady
            {
                IsReady = !currentReady
            });
        }

        public void StartGame()
        {
            if (NetClient.CurrentRoom == null)
            {
                AppendLog("开始游戏跳过: 当前不在房间内。");
                return;
            }

            NetClient.Send(new C2S_StartGame());
        }

        public void LeaveRoom()
        {
            if (NetClient.CurrentRoom == null)
            {
                AppendLog("离开房间跳过: 当前不在房间内。");
                return;
            }

            NetClient.Send(new C2S_LeaveRoom());
        }

        public void SendRoomNotice()
        {
            if (NetClient.CurrentRoom == null)
            {
                AppendLog("发送房间通知跳过: 当前不在房间内。");
                return;
            }

            NetClient.Send(new C2S_SendRoomNotice
            {
                Content = roomNoticeContent
            });
        }

        private void OnLoginResult(S2C_LoginResult msg)
        {
            AppendLog($"登录结果 success={msg.Success} reason={msg.Reason}");
        }

        private void OnAnnouncementSync(S2C_AnnouncementSync msg)
        {
            AppendLog($"公告同步 sender={msg.SenderName} content={msg.Content}");
        }

        private void OnRoomListResponse(S2C_RoomListResponse msg)
        {
            if (msg == null || msg.Rooms == null || msg.Rooms.Length == 0)
            {
                _roomListText = "(当前没有可加入的房间)";
                AppendLog("房间列表返回: 当前没有房间。");
                return;
            }

            var sb = new StringBuilder();
            for (int i = 0; i < msg.Rooms.Length; i++)
            {
                RoomBriefInfo room = msg.Rooms[i];
                if (room == null)
                {
                    continue;
                }

                sb.AppendLine($"{i + 1}. 房间Id={room.RoomId} 名称={room.RoomName} 人数={room.MemberCount}/{room.MaxMembers} 状态={room.State}");
            }

            _roomListText = sb.ToString().TrimEnd();
            if (string.IsNullOrWhiteSpace(joinRoomId) && msg.Rooms[0] != null)
            {
                joinRoomId = msg.Rooms[0].RoomId;
            }

            AppendLog($"房间列表返回: count={msg.Rooms.Length}");
        }

        private void OnCreateRoomResult(S2C_CreateRoomResult msg)
        {
            AppendLog($"建房结果 success={msg.Success} roomId={msg.RoomId} reason={msg.Reason}");
            if (msg != null && msg.Success)
            {
                joinRoomId = msg.RoomId;
            }
        }

        private void OnJoinRoomResult(S2C_JoinRoomResult msg)
        {
            AppendLog($"加房结果 success={msg.Success} roomId={msg.RoomId} reason={msg.Reason}");
        }

        private void OnLeaveRoomResult(S2C_LeaveRoomResult msg)
        {
            AppendLog($"离房结果 success={msg.Success}");
            _registeredRoomId = string.Empty;
            _roomStateText = "(当前不在房间内)";
        }

        private void OnRoomSetupResult(S2C_RoomSetupResult msg)
        {
            AppendLog($"进房确认 success={msg.Success} roomId={msg.RoomId} reason={msg.Reason}");

            if (!msg.Success || NetClient.CurrentRoom == null)
            {
                return;
            }

            string roomId = NetClient.CurrentRoom.RoomId;
            if (_registeredRoomId == roomId)
            {
                return;
            }

            _registeredRoomId = roomId;

            RegisterCurrentRoomEvents();
            RefreshCurrentRoomState();
        }

        private void OnRoomNoticeSync(S2C_RoomNoticeSync msg)
        {
            AppendLog($"房间通知 sender={msg.SenderSessionId} content={msg.Content}");
        }

        private void OnRoomSnapshot(S2C_RoomSnapshot msg)
        {
            RefreshCurrentRoomState();
        }

        private void OnMemberJoined(S2C_MemberJoined msg)
        {
            AppendLog($"成员加入 session={msg?.Member?.SessionId}");
            RefreshCurrentRoomState();
        }

        private void OnMemberLeft(S2C_MemberLeft msg)
        {
            AppendLog($"成员离开 session={msg?.SessionId}");
            RefreshCurrentRoomState();
        }

        private void OnMemberReadyChanged(S2C_MemberReadyChanged msg)
        {
            AppendLog($"准备状态变化 session={msg.SessionId} ready={msg.IsReady}");
            RefreshCurrentRoomState();
        }

        private void OnGameStarted(S2C_GameStarted msg)
        {
            AppendLog($"游戏开始 startUnixTime={msg.StartUnixTime}");
            RefreshCurrentRoomState();
        }

        private void OnGameEnded(S2C_GameEnded msg)
        {
            AppendLog($"游戏结束 winnerOrReason={msg.WinnerSessionId}");
            RefreshCurrentRoomState();
        }

        private void RegisterCurrentRoomEvents()
        {
            if (NetClient.CurrentRoom == null)
            {
                return;
            }

            NetClient.CurrentRoom.NetEventSystem.Register<S2C_RoomSnapshot>(OnRoomSnapshot).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_MemberJoined>(OnMemberJoined).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_MemberLeft>(OnMemberLeft).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_MemberReadyChanged>(OnMemberReadyChanged).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_GameStarted>(OnGameStarted).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_GameEnded>(OnGameEnded).UnRegisterWhenMonoDisable(this);
            NetClient.CurrentRoom.NetEventSystem.Register<S2C_RoomNoticeSync>(OnRoomNoticeSync).UnRegisterWhenMonoDisable(this);
        }

        private void RefreshCurrentRoomState()
        {
            if (NetClient.CurrentRoom == null)
            {
                _roomStateText = "(当前不在房间内)";
                return;
            }

            ClientRoomSettingsComponent settings = GetRoomSettings();
            if (settings == null)
            {
                _roomStateText = $"房间Id: {NetClient.CurrentRoom.RoomId}\n(RoomSettings 组件尚未就绪)";
                return;
            }

            var sb = new StringBuilder();
            sb.AppendLine($"房间Id: {NetClient.CurrentRoom.RoomId}");
            sb.AppendLine($"房间名: {settings.RoomName}");
            sb.AppendLine($"人数: {settings.Members.Count}/{settings.MaxMembers}");
            sb.AppendLine($"是否已开局: {settings.IsGameStarted}");
            sb.AppendLine("成员列表:");

            string selfSessionId = NetClient.Session != null ? NetClient.Session.SessionId : string.Empty;
            foreach (var kv in settings.Members)
            {
                MemberInfo member = kv.Value;
                if (member == null)
                {
                    continue;
                }

                string selfTag = member.SessionId == selfSessionId ? " [自己]" : string.Empty;
                string ownerTag = member.IsOwner ? " [房主]" : string.Empty;
                string readyTag = member.IsReady ? " [已准备]" : " [未准备]";
                sb.AppendLine($"- {member.DisplayName} ({member.SessionId}){selfTag}{ownerTag}{readyTag}");
            }

            _roomStateText = sb.ToString().TrimEnd();
        }

        private ClientRoomSettingsComponent GetRoomSettings()
        {
            return NetClient.CurrentRoom != null
                ? NetClient.CurrentRoom.GetComponent<ClientRoomSettingsComponent>()
                : null;
        }
    }
}

## 12. 第九步：为什么房间组件不是写完就能生效

这是最容易踩坑的一步。

你写了：

- `[RoomComponent(930, "RoomNotice", ...)]`

并不代表所有房间都会自动带 `RoomNotice`。

房间组件要真正生效，必须出现在某个房间的 `ComponentIds` 里。

最小做法就是像本文这样，在建房时直接写：

```csharp
ComponentIds = new[]
{
    ComponentIdConst.RoomSettings,
    ComponentIdConst.RoomNotice
}
```

也就是说：

- 全局模块靠注册自动生效
- 房间业务组件靠“注册 + 挂进房间”才生效

## 13. 第十步：新增代码后必须重新生成

当你新增了下面任意内容后，都必须重新生成：

- `[NetMsg]`
- `[ServerModule]`
- `[ClientModule]`
- `[RoomComponent]`
- `[NetHandler]`

你至少要重新生成出这些内容：

- `MsgIdConst`
- `ComponentIdConst`
- 自动注册表
- 自动绑定器

菜单入口：

- `StellarNetLite/重新生成协议与组件常量表`
- `StellarNetLite/强制重新生成协议与组件常量表`

如果你忘了这一步，最常见的表现就是：

- 消息发得出去但收不到
- 新组件写了但房间里没有
- 常量表里没有新协议 Id / 组件 Id

## 15. 第十二步：最小验证顺序

如果你只开一个客户端实例，能验证：

- 登录
- 全局公告
- 建房
- 进房
- 房间通知

但如果你想把“房间列表 / 加房 / 双人准备 / 开局”也完整走通，最少要双开两个实例。

推荐的最小验证方式是：

1. 一个实例用 `Host` 模式启动，挂 `BasicNetLauncher + BasicClientConsole`
2. 另一个实例用 `Client` 模式启动，也挂 `BasicNetLauncher + BasicClientConsole`
3. 两边先都按 `L` 登录
4. `Host` 端按 `3` 建房，记住自动显示出来的 `RoomId`
5. 纯客户端按 `2` 拉房间列表，确认房间列表里能看到刚建的房间
6. 纯客户端把 `JoinRoomId` 填成该房间 Id，再按 `4` 加房
7. 观察两边都收到房间快照，且“当前房间状态”区域里都能看到两名成员
8. 两边分别按 `R`，观察准备状态变化
9. 房主端按 `S`，观察两边都收到 `S2C_GameStarted`
10. 任意一边按 `5` 发房间通知，观察双方都收到 `S2C_RoomNoticeSync`
11. 纯客户端按 `Esc` 离房，再按 `2` 拉房间列表，确认成员数发生变化

只要这套双实例最小验证能跑通，就说明下面这些点都已经通了：

## 16. 你最容易写错的地方

### 16.1 把协议放错目录

协议一定放 `Shared`。

### 16.2 把全局功能写成房间组件

结果就是功能离开房间就失效。

### 16.3 把房间功能写成全局模块

结果就是拿不到 `Room` 上下文。

### 16.4 忘了把房间组件挂进 `ComponentIds`

这会导致组件根本不被创建。

### 16.5 忘了重新生成

这是最常见的“代码都写了但没反应”来源。

### 16.6 在 `S2C_RoomSetupResult.Success == true` 之前急着发房间消息

当前框架会在最终确认前拦截 `NetScope.Room` 发送。

### 16.7 客户端额外包一层业务本地事件

不推荐。

推荐直接广播协议对象本身，让业务层直接监听协议对象。

### 16.8 只开一个实例就想验证“加房”

这不是框架问题，而是验证方式不对。

下面这些链路天然更适合双实例验证：

- 拉房间列表
- 玩家 B 加入玩家 A 创建的房间
- 双人准备
- 房主开局

所以本文现在把最小验证流程明确拆成了：

- 单实例可验证的部分
- 双实例才能完整验证的部分

在不依赖任何 Demo 的前提下，本文演示的开发主线就是：
